To Do App

Overview

A WinForms task management application built with my duo partner.

Users can create an account or log in, then create, edit, export and organize their tasks efficiently. The app helps users stay on top of their daily goals with a simple and intuitive interface.

Technologies Used

  • WinForms C#

Features

  • User account creation and login
  • Add, edit, and remove tasks
  • Export tasks for backup or sharing
  • Dynamic task updates
